# Installation

Install the library and its peer dependencies in your React project:

<Tabs groupId="package-manager">


  </TabItem>
  <TabItem value="pnpm" label="pnpm">

  </TabItem>
  <TabItem value="yarn" label="yarn">

  </TabItem>
  <TabItem value="bun" label="bun">

```bash

```

  </TabItem>

React Kit has peer dependencies that your project should satisfy (they are often already present in MUI-based apps):

- `react` and `react-dom` (^18.3.1 or ^19.0.0)
- `@mui/material` (^7.1.0)
- Optional: `react-router-dom`, `@mui/x-data-grid`, `date-fns` for specific components

If you start from a minimal React app, install MUI and React Router as well:

## Usage example

After installing React Kit, import and use components or utilities in your app:

**Using a component**

```tsx
import { SuccessButton } from '@js-smart/react-kit';

function MyForm() {
	return <SuccessButton onClick={() => handleSave()}>Save</SuccessButton>;
}
```

**Using a utility**

```tsx
import { isBlankOrEmpty } from '@js-smart/react-kit';

const isEmpty = isBlankOrEmpty(input); // true if null, undefined, or blank string
```

For more examples, see the [Introduction](/docs/introduction) page or the [react-kit-demo](https://github.com/js-smart/react-kit/tree/main/apps/react-kit-demo) app.

## Troubleshooting

**Peer dependency warnings**

If you see peer dependency warnings for `@mui/material`, `react`, or `react-dom`, install the versions React Kit expects (see the list above). For example:

```bash
pnpm add react@^19 react-dom@^19 @mui/material@^7 @emotion/react @emotion/styled
```

**"Module not found" or import errors**

- Ensure the package is installed: `@js-smart/react-kit` appears in your `package.json` dependencies.
- Restart the dev server and clear any cache (e.g. delete `node_modules/.cache` if present).

**Styles or MUI theme not applied**

React Kit components rely on MUI. Wrap your app (or the part that uses React Kit) with `ThemeProvider` from `@mui/material` and ensure MUI's CSS is loaded. See [MUI installation](https://mui.com/material-ui/getting-started/installation/).

**TypeScript errors**

Ensure your project uses TypeScript 5.x and that `@types/react` and `@types/react-dom` are compatible with your React version (18.3+ or 19.x).
